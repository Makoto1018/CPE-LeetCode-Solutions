### Problem Description

A single number is a number that appeared only once in the MyNumbers table.

Find the largest single number. If there is no single number, report null.

### code
```sql
WITH n as (SELECT num
    FROM MyNumbers
    GROUP BY num
    having count(*)=1
)
SELECT max(num) as num
FROM n;
```
